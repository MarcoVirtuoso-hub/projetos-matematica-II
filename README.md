# 🎯 Projeto de Objetivos

Projeto interativo desenvolvido com HTML, CSS e JavaScript para acompanhar metas pessoais através de cronômetros regressivos.

---

## 🚀 Funcionalidades

- Sistema de abas interativas
- Cronômetros em tempo real
- Layout responsivo
- Interface moderna
- Navegação dinâmica com JavaScript

---

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## 📌 Objetivos do projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- manipulação do DOM;
- estilização com CSS;
- lógica em JavaScript;
- responsividade;
- publicação com GitHub Pages.
